About DetMeter

Version: 1.0.0

Description: Plugin to compare and measure SIEM detection effectiveness for MITRE Caldera operations.

How It Works

  1. Select a completed operation from Caldera
  2. DetMeter extracts MITRE ATT&CK techniques used
  3. Queries configured SIEM systems for detection events
  4. Compares detection rates across SIEMs
  5. Displays results in charts and tables

Endpoints

  • GET /detmeter - This interface
  • POST /detmeter/api/analyze - Analyze operation
  • GET /detmeter/api/status - Check SIEM status
  • GET /detmeter/api/operations - List operations
